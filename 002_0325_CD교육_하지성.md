## 3. 🖥️ VS Code, 터미널로 열기

### Step 1 — 작업 폴더로 이동
```powershell
# 탐색기에서 경로 Ctrl+C 후 붙여넣기
cd C:\작업폴더\경로
ls
```

### Step 2 — 가상환경 확인 및 활성화
```powershell
conda env list                  # 설치된 가상환경 목록 확인
conda activate (가상환경명)      # 원하는 환경 활성화
```

### Step 3 — VS Code 실행
```powershell
code .                          # 현재 폴더를 VS Code로 열기
```

✓ 목적(Task): LLM이 수행해야 할 핵심 과업
✓ 문맥정보(Context): 과업이 필요한 목적과 배경 정보
✓ 지시사항(Instructions): 결과물의 형식/조건/세부지시
✓ 단순 지시(Task)만으로도 답변을 생성할 수 있으나, Context와
Instructions를 추가하면 결과의 정확성과 만족도가 높아짐
