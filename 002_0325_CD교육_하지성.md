## 프롬프트 핵심 3요소
✓ 목적 + 문맥정보 + 지시사항
✓ 목적(Task): LLM이 수행해야 할 핵심 과업
✓ 문맥정보(Context): 과업이 필요한 목적과 배경 정보
✓ 지시사항(Instructions): 결과물의 형식/조건/세부지시
✓ 단순 지시(Task)만으로도 답변을 생성할 수 있으나, Context와
Instructions를 추가하면 결과의 정확성과 만족도가 높아짐

## 프롬프트 Context(맥락/배경 제공)
✓ 지시한 과업이 왜 필요한지, 어떤 상황에서 요청하는지 설명
✓ LLM이 사용자의 목적과 배경을 이해하여 더 정교하고 실행 가능한 결
과를 얻을 수 있음
✓ 맥락은 답변의 방향성과 현실성을 높이며 단순 아이디어가 아닌, 실제
문제를 반영한 해결책이 도출될 가능성을 높임
✓ (예시 프롬프트) “현재 노인 인구가 증가하고 은퇴 시기 또한 점점 늦춰
지는 상황이 발생하고 있어. 이에 따라 정년 연장, 노인 취업, 복지 예산
등 다양한 분야에서 새로운 정책이 나와야한다는 분위기가 조성되고
있어. 이와 관련하여 노인 정책 입안을 위한 아이디어를 제시해줘”



  


code .
